useLiveCanvas

  • useLiveCanvas(uniqueKey: string, canvasElementRef: string | RefObject<null | HTMLElement>, active?: boolean, tool?: InkingTool, lineBrush?: IBrush, offset?: IPoint, scale?: number, referencePoint?: CanvasReferencePoint, isCursorShared?: boolean, localUserPictureUrl?: string): IUseLiveCanvasResults
    • Defined in packages/live-share-react/src/live-hooks/useLiveCanvas.ts:38

    React hook for using a Live Share Canvas LiveCanvas and InkingManager.

    remarks

    Use this hook to set up an LiveCanvas object, which is used for collaborative inking. It takes several input parameters, including a unique key, a reference to a canvas element, and various settings for the inking tool, brush, and canvas offset and scale.

    Parameters

    • uniqueKey: string

      the unique key for the LiveCanvas. If one does not yet exist, a new one.

    • canvasElementRef: string | RefObject<null | HTMLElement>

      the HTML div element ref or document ID that InkingManager will use for canvas-based collaboration.

    • Optional active: boolean

      Optional. Stateful boolean that will activate/de-activate InkingManager accordingly.

    • Optional tool: InkingTool

      Optional. Stateful enum for what tool to use in the InkingManager.

    • Optional lineBrush: IBrush

      Optional. Stateful lineBrush object for the selected lineBrush options to use in InkingManager.

    • Optional offset: IPoint

      Optional. Stateful offset point to use in the InkingManager. Gets the viewport offset. Defaults to 0,0.

    • Optional scale: number

      Optional. Stateful scale number to use in the InkingManager. Defaults to 1 and must be greater than 0.

    • Optional referencePoint: CanvasReferencePoint

      Optional. Stateful reference point enum to use in the InkingManger. Defaults to "center".

    • Optional isCursorShared: boolean

      Optional. Stateful boolean flag for whether cursor should be shared in LiveCanvas. Defaults to false.

    • Optional localUserPictureUrl: string

      Optional. url string for the local user to display alongside their cursor. Defaults to undefined.

    Returns IUseLiveCanvasResults

    IUseLiveCanvasResults object that contains the liveCanvas data object and inkingManager.

    React hook for using a Live Share LivePresence.

    remarks

    Use this hook if you want to track presence for who is currently using this component, such as who is online or who is viewing a specific document. With presence, you can sent along any custom user data. This is useful for rendering a list of users, profile pictures, cursor positions, and more.

    • TData extends object = object

      Optional typing for the custom user presence data object. Default is object type.

    Parameters

    • Optional initialData: TData

      Optional. Initial presence data object for the user.

    • initialPresenceState: PresenceState = PresenceState.online

      Optional. Initial status of the user's presence. Default is online.

    • uniqueKey: string = ":<dds-default>"

      Optional. The unique key for LivePresence. If one does not yet exist, a new one will be created, otherwise it will use the existing one. Default value is ":"

    Returns IUseLivePresenceResults<TData>

    stateful localUser, otherUsers list, and allUsers list. Also returns a callback method to update the local user's presence and the LivePresence Fluid object.
